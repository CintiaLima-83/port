# 💼 Portfólio Pessoal

Portfólio desenvolvido com **React** para apresentar meus projetos, habilidades e informações de contato.

O objetivo deste projeto é reunir os principais trabalhos desenvolvidos durante minha formação, demonstrando a evolução dos conhecimentos adquiridos em desenvolvimento web, com foco em interfaces modernas, responsivas e boas práticas de programação.

## 🚀 Tecnologias Utilizadas

* React
* JavaScript
* Next.js
* TypeScript
* CSS Modules
* Git
* GitHub

## 📂 Projetos Apresentados

### 🌍 Portal de Viagem

Aplicação desenvolvida com Next.js para apresentação de destinos turísticos, utilizando navegação entre páginas, rotas dinâmicas e design responsivo.

**Tecnologias:** Next.js, React, CSS Modules

---

### 🎬 CineList

Aplicação para busca de filmes utilizando a API do TMDB, permitindo pesquisar títulos e visualizar informações de forma dinâmica.

**Tecnologias:** Next.js, TypeScript, API TMDB

---

### 📱 Landing Page iPhone 17

Landing page inspirada na apresentação de produtos da Apple, com foco em design moderno, responsividade e experiência visual.

**Tecnologias:** React, JavaScript e CSS

## 🛠️ Como Executar o Projeto

Clone este repositório:

```bash
git clone https://github.com/CintiaLima-83/port
```

Acesse a pasta do projeto:

```bash
cd meu-portfolio
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

O projeto estará disponível em:

```txt
http://localhost:5173
```
